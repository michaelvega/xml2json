XML2JSON API Homepage

API uses xml2js to convert xml strings to JSON

  • Input: XML String
  • Output: JSON file with 4 space tabs (if available)

Site Map

  • /converter - POST request the XML string

Start Guide

To upload your own XML files, POST request with an XML string. For Google Drive:

  • Read XML as utf8: xmlString
  • Request the site with these parameters:
  • uri: url/converter,
  • body: xmlString,
  • method: 'POST',
  • headers: {  'Content-Type': 'text/plain'  }
